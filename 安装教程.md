# DiscordAITranslator 安装教程

## 安装前准备

请先确认已安装：

- Discord
- BetterDiscord

## 一、安装插件

把 `DiscordAITranslator.plugin.js` 放到下面目录：

```text
%AppData%\BetterDiscord\plugins
```

然后打开 Discord -> `设置` -> `BetterDiscord` -> `插件`，启用：

```text
DiscordAITranslator
```

## 二、安装 BDFDB Library

本插件依赖 `BDFDB Library`。

下载地址：

[https://mwittrien.github.io/downloader/?library](https://mwittrien.github.io/downloader/?library)

下载完成后，同样放到：

```text
%AppData%\BetterDiscord\plugins
```

## 三、安装 system24 主题（可选）

### 方式 A：主题文件安装

把 `system24.theme.css` 放到：

```text
%AppData%\BetterDiscord\themes
```

然后在 BetterDiscord 主题页启用 `system24`。

### 方式 B：直链导入安装

推荐直接使用下面的导入地址：

```text
https://refact0r.github.io/system24/build/system24.css
```

相关页面：

- system24 官方说明页：
  [https://refact0r.github.io/system24/](https://refact0r.github.io/system24/)
- system24 主题文件页：
  [https://github.com/refact0r/system24/blob/main/theme/system24.theme.css](https://github.com/refact0r/system24/blob/main/theme/system24.theme.css)

## 四、首次配置插件

1. 打开 `DiscordAITranslator` 设置
2. 选择主翻译服务商
3. 填写 API Key
4. 服务商要求时继续填写 Endpoint 和 Model
5. 设置发送 / 收到消息的语言
6. 按需开启收到消息自动翻译与高级功能

## 五、0.3.26 版本特别说明

这一轮升级后，建议重点检查下面几项：

- 设置页点击 `Select` / 输入框时是否还会跳顶
- 发送消息目标语言和源语言相同的时候，是否直接发送原文
- 收到消息目标语言和消息主体语言相同的时候，是否不再自动生成同语言改写
- 输入框总翻译开关关闭后，单条消息翻译按钮是否仍可用
- 自动翻译或历史补翻后，页面是否仍能保持在原来的观看位置

当前工作区还补充了旧缓存防回归逻辑：

- 旧版已经缓存的同语言改写译文不应继续复用
- 插件重新渲染消息时会自动清掉这类不该保留的旧缓存结果

## 六、替换新版本后的建议操作

1. 替换新的 `DiscordAITranslator.plugin.js`
2. 回到 Discord 插件页重新启用一次插件，或直接按 `Ctrl+R`
3. 回到常用频道，重点检查上面的 5 个测试点

## 七、常见问题

### 1. 插件没有显示

检查 `DiscordAITranslator.plugin.js` 是否放在：

```text
%AppData%\BetterDiscord\plugins
```

### 2. 提示缺少依赖

说明 `BDFDB Library` 未安装，按上面的链接下载安装即可。

### 3. 修改后没有生效

在 Discord 里按一次：

```text
Ctrl+R
```

重新加载客户端。
